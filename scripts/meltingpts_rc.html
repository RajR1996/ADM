STORED PROCEDURE EXAMPLE

This form returns a list of elements with a melting point within a user-defined range.


The PL/SQL stored procedure meltingpts_rc is called using :meltlow,:melthigh, and :cursbv as parameters.


The :cursbv parameter returns a pointer to a ref_cursor, which is then 'executed' to retrieve the data.
