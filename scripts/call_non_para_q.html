NON-PARAMETERISED QUERY Example

This form returns rows from the EMP table which match the specified job type.


The form accepts the input value, posts it to a session variable $_POST['job'], then launches non_para_q_emp.php

Next, para_q_emp.php reads $_POST['job'] into the PHP variable $job

The SQL statement contains $job embedded directly as a character string.

Oracle parses the entire statement including the user-input character string (which might change the meaning of the SQL), this is the risk of SQL Injection.
