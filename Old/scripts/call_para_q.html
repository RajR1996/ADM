PARAMETERISED QUERY EXAMPLE

This form returns rows from the EMP table which match the specified job type.


The form accepts the input value, posts it to a session variable $_POST['job'], then launches para_q_emp.php

Next, para_q_emp.php reads $_POST['job'] into the PHP variable $job, and later binds $job to the Oracle bind variable :job

The SQL prepared statement contains :job as a parameter-token. Oracle parses this separately, and the parameter value is inserted later, . .
so the user-input value is not interpreted, avoiding the risk of SQL Injection
